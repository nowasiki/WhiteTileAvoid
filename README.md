# 别踩白块

## 简介

本游戏采用C/C++编写，是我学习C语言的第二个项目。

## 游玩方式

通过点击最下方的黑方块，以获得积分。

## Download

在`release`中找到`WhiteTileAvoid.exe`文件下载游玩即可。
